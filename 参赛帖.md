## 1. Demo 简介

**是什么：** 声浪雷达是一个 Web 应用，支持粘贴 B站视频链接一键分析评论情感，或输入话题关键词全网搜索聚合分析。全部分析在本地完成，数据不出本机。

**面向谁：**
- 内容创作者 / UP主 / 博主 — 复盘视频真实口碑，找出观众真正关心的问题
- 品牌运营与公关团队 — 新品视频发布后快速判断用户是被种草还是被劝退
- 市场研究人员 — 从评论中提炼用户语言和消费动机，辅助产品决策
- 校园社团与组织传播岗 — 活动视频发布后快速了解受众反馈
- 公共服务部门与机构 — 了解公众对政策宣传内容的真实反馈

**核心功能：**

**视频即时分析** — 粘贴 B站视频链接，自动抓取评论并进行情感分析、风险识别、关键词提取、观点聚类，生成 AI 舆情报告。把原本几个小时的"人工翻评论"压缩到分钟级 AI 自动洞察。

**话题全网搜索** — 输入话题关键词（如"AI大模型"、"新能源"），自动搜索相关视频，聚合 Top N 视频评论进行整体舆情分析，支持跨视频对比，看清某个话题在不同视频下的舆论差异。

**Agent 智能预警** — 采用四阶段 Agent 架构：智能预警（LLM 对比历史基线判定舆情风险，替代固定阈值）、归因分析（ReAct 多轮推理自主查证评论细节）、策略生成（输出回复草稿、置顶推荐等可执行建议）、多视频协同分析。分析报告下方可展开查看 Agent 完整推理轨迹，每一步"思考→查证→结论"都透明可追溯。

## 2. Demo 创作思路

**灵感来源：**

短视频和内容社区里沉淀了大量公开评论与弹幕，但内容创作者、品牌运营、市场研究、公关和校园组织几乎只能靠人工翻评论，难以快速看清"用户在夸什么、在骂什么、争议点在哪、负面是否正在升温"。我自己在做内容和参与社团活动时，经常被一条爆款视频下几千条评论"淹没"，凭手感判断舆论既慢又不准。我希望用 AI 把这些散乱的评论整理成一份可读、可解释的舆情洞察。

**想解决的问题：**
- 没有好用的免费工具时，用户只能逐条翻评论或刷弹幕，效率低、覆盖不足
- 情绪和观点全靠主观判断，容易漏掉负面集中爆发的早期信号
- 固定阈值告警误报率高，不同 UP 主的评论基线差异很大
- 现有 SaaS 工具需要上传数据到云端，存在隐私顾虑

**为什么做这个方向：**

选择 B站作为切入点，是因为 B站评论区数据公开可获取、用户群体活跃、讨论质量高。技术上，B站近年启用了 WBI 强制签名机制，大部分第三方工具已经无法抓取评论，这恰好构成了技术壁垒。在 AI 方面，选择了 Agent 架构而非简单的 LLM 单次调用，让系统能自主查证、修正误判（比如识别反讽评论），输出更可靠的分析结果。后续可扩展小红书等平台支持，以及团队协作和开放 API。

## 4. TRAE 实践过程

整个项目用 TRAE 从零开发，以下是完整的开发流程：

**第一步：基础框架搭建**

用 TRAE 搭建了 Python 后端（标准库 http.server，零框架依赖）+ Vue 3 前端（Vite 6 构建）的基础架构。实现了 B站视频链接解析（支持 b23.tv 短链和 BV 长链）、评论抓取、基础情感分析（三库合并词典法：Hownet + NTUSD + 清华李军，约 2.4 万词，含程度词加权和否定词翻转）。

**第二步：核心功能完善**

实现了完整的舆情分析链路：情感分析、jieba TF-IDF 关键词提取、观点聚类、风险识别（自动标记辱骂、引战、敏感话题）、AI 报告生成（支持 DeepSeek/智谱/通义千问/OpenAI/Ollama 多个 LLM 提供商，无 Key 时自动降级为模板报告）。前端实现了 ECharts 情感饼图、暗色模式、响应式布局、骨架屏加载态、Design System（克莱因蓝品牌色、8 级灰阶、8px 间距体系）。

**第三步：话题雷达**

实现了 B站 WBI 签名搜索接口对接，支持输入话题关键词全网搜索相关视频，取 Top N 热门视频分别抓取评论后聚合分析，生成跨视频对比的话题舆情报告。

**第四步：Agent 智能系统**

实现了四阶段 Agent 架构：
- 智能预警：LLM 对比历史基线判定是否告警，替代固定阈值，大幅降低误报率
- 归因分析：ReAct 循环多轮推理，Agent 自主调用工具（搜索评论、修正分类、查看趋势、查看聚类详情）查证分析
- 策略生成：输出结构化行动建议（回复草稿可一键复制、置顶推荐、监控建议、团队通知）
- 多视频协同：跨视频情绪趋势对比、UP 主历史舆情追踪

**第五步：工程完善**

补充了 Docker 多阶段构建、Railway 云部署适配、Webhook 告警推送（飞书/钉钉/企业微信）、单元测试覆盖、完整的隐私安全设计（本地分析不经过第三方、二维码本地生成、SSRF 防护）。
